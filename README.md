1. 📌 Project Title

Netflix Data Analysis (EDA)

2. 🧠 Description

This project is an exploratory data analysis (EDA) of Netflix’s catalog using Python. It focuses on data cleaning, transformation, and visualization to understand content composition. It identifies patterns in movies vs TV shows, trends over time, and top content-producing countries. Tools: Pandas, Matplotlib, Jupyter Notebook.

3. 🎯 Objective

This project aims to perform exploratory data analysis (EDA) on the Netflix dataset to better understand movie trends from the 1990s. The goal is to identify the most frequent movie duration during that decade and analyze the number of short action movies (under 90 minutes), providing insights into content patterns of that period.

4. 📊 Dataset

Netflix dataset provided by DataCamp

5. 🛠️ Tools used:

Python
Pandas
Matplotlib
Jupyter Notebook

6. 📈 Results

This exploratory data analysis of Netflix movies from the 1990s indicates that the most common movie duration is around 100 minutes.
A histogram was generated to visualize the distribution of movie lengths, revealing a concentration between 90 and 120 minutes.
Furthermore, the analysis identified the number of short action movies (under 90 minutes) by filtering the dataset and applying conditional counting techniques.

7. 🚀 Conclusion

This exploratory data analysis provides insights into Netflix movies from the 1990s. The results indicate that the most frequent movie duration is approximately 100 minutes, with most films concentrated between 90 and 120 minutes. Additionally, short action movies (under 90 minutes) are relatively uncommon.
A histogram was created using Matplotlib to visually represent the distribution of movie durations, allowing for a clearer understanding of the data patterns. Overall, this analysis helps to better understand content distribution and trends in movie production during the 1990s.
